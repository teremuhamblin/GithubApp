# Index des Patterns — GithubApp

Bienvenue dans le dossier `pattern/` du projet GithubApp.  
Ce répertoire regroupe toutes les conventions, normes et standards utilisés dans le projet.

---

## 📁 Contenu du dossier

### 1. Identité visuelle
- `identitée-visuelle.md`  
Décrit le style graphique, le logo, les couleurs et l’identité cyber‑tech du projet.

### 2. Conventions de nommage
- `naming-convention.md`  
Règles pour nommer fichiers, dossiers, variables, fonctions et constantes.

### 3. Style de commits
- `commit-style.md`  
Normes basées sur Conventional Commits pour garder un historique propre.

### 4. Design Patterns
- `design-patterns.md`  
Architecture interne, patterns utilisés, modularité, handlers, utils, etc.
